# Imágenes premio

Copia aquí las 3 fotos que se irán desbloqueando por trozos a medida que
el grupo suma puntos, con estos nombres exactos:

- `premio1.jpg`
- `premio2.jpg`
- `premio3.jpg`

Si prefieres otros nombres o extensión (`.png`, `.webp`...), cámbialos en
`PRIZE_IMAGES` dentro de `js/challenges.js`.

Mientras estas imágenes no existan, la pestaña "Premios" de la app muestra
un placeholder ("Imagen pendiente") en su lugar — no rompe nada, solo
actualiza este README/estas fotos cuando las tengas listas.
